# 🎨 Цветовая схема текста - Документация

## Обновлённая палитра

### Основные цвета текста

| Переменная | Значение | Использование |
|------------|----------|---------------|
| `--text-primary` | `#1a202c` | Заголовки, основной текст |
| `--text-secondary` | `#4a5568` | Основной текст карточек |
| `--text-muted` | `#718096` | Второстепенная информация |
| `--text-light` | `#a0aec0` | Светлый текст (метки) |
| `--text-white` | `#ffffff` | Белый текст |
| `--text-inverse` | `#f7fafc` | Текст на тёмном фоне |

### Цвета для ссылок

| Переменная | Значение | Использование |
|------------|----------|---------------|
| `--link-color` | `#667eea` | Обычные ссылки |
| `--link-hover` | `#764ba2` | Ссылки при наведении |

---

## 📊 Иерархия текста

### 1. Заголовки (h1-h6)
```css
color: var(--text-primary);  /* #1a202c - очень тёмный */
font-weight: 700;
```

### 2. Основной текст
```css
p, .card-text {
    color: var(--text-secondary);  /* #4a5568 - тёмно-серый */
}
```

### 3. Второстепенный текст
```css
.text-muted, small {
    color: var(--text-muted);  /* #718096 - серый */
}
```

### 4. Метки и подсказки
```css
.text-light {
    color: var(--text-light);  /* #a0aec0 - светло-серый */
}
```

## 🎯 Контексты использования

### Светлый фон (карточки, формы)
```
Фон: #ffffff
Текст: #1a202c (primary)
Вторичный: #4a5568 (secondary)
```

### Тёмный фон (hero, footer)
```
Фон: linear-gradient(#667eea, #764ba2)
Текст: #f7fafc (inverse)
Вторичный: rgba(255,255,255,0.9)
```

### Навигация
```
Фон: rgba(255,255,255,0.95)
Текст: #1a202c (primary)
Активный: #667eea (primary-color)
```

## 🌓 Тёмная тема (auto)

Автоматически включается через `prefers-color-scheme: dark`

### Изменения в тёмной теме:

| Элемент | Светлая | Тёмная |
|---------|---------|--------|
| **Основной текст** | #1a202c | #f7fafc |
| **Вторичный** | #4a5568 | #e2e8f0 |
| **Приглушённый** | #718096 | #a0aec0 |
| **Фон карточек** | #ffffff | #2d3748 |
| **Границы** | #e2e8f0 | #4a5568 |

## ✅ Контрастность (WCAG 2.1)

Все цвета соответствуют уровню **AA**:

| Комбинация | Контраст | Стандарт |
|------------|----------|----------|
| primary на white | 12.6:1 | ✅ AAA |
| secondary на white | 7.8:1 | ✅ AA |
| inverse на gradient | 11.2:1 | ✅ AAA |
| link на white | 8.9:1 | ✅ AA |

## 🎨 Примеры использования

### 1. Карточка слова
```html
<div class="card">
    <h5 class="card-title text-primary">Слово</h5>
    <p class="card-text text-secondary">Значение...</p>
    <small class="text-muted">1990-е</small>
</div>
```

### 2. Hero секция
```html
<div class="hero-section">
    <h1 class="text-white">Заголовок</h1>
    <p class="lead" style="color: rgba(255,255,255,0.9)">Описание</p>
</div>
```

### 3. Навигация
```html
<nav class="navbar">
    <a class="nav-link">Основная</a>
    <a class="nav-link active">Активная</a>
</nav>
```

## 🔧 Настройка

### Изменить основной цвет текста:
```css
:root {
    --text-primary: #YOUR_COLOR;
}
```

### Изменить цвет ссылок:
```css
:root {
    --link-color: #YOUR_COLOR;
    --link-hover: #YOUR_HOVER_COLOR;
}
```

### Отключить тёмную тему:
```css
/* Удалить или закомментировать */
@media (prefers-color-scheme: dark) {
    /* ... */
}
```

---

## 📱 Адаптивность

Цвета автоматически адаптируются:

- **Дневной режим** - тёмный текст на светлом фоне
- **Ночной режим** - светлый текст на тёмном фоне
- **Hero/Footer** - всегда светлый текст на тёмном
