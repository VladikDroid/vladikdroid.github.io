# 🎨 БРИГАДА.РУ - Уличный стиль 90-х

## 🎯 Концепция дизайна

**Бригада.ру** — это словарь молодёжного сленга в стиле культового сериала "Бригада" (2002).

### Визуальная философия:
- 🌑 **Тёмная тема** — как ночь на улицах 90-х
- 💚 **Неоновый зелёный** — легендарные лампасы Adidas
- 🧡 **Ярко-оранжевый** — предупреждающие знаки, семки
- ⚡ **Глитч-эффекты** — эра кассет и помех
- 🔠 **Грубые шрифты** — стиль уличных вывесок

## 🎨 Цветовая палитра

### Основные цвета

| Цвет | Значение | Использование |
|------|----------|---------------|
| `#0a0a0a` | Глубокий чёрный | Основной фон |
| `#141414` | Тёмно-серый | Фон карточек |
| `#39ff14` | **Неоновый зелёный** | Акценты, кнопки |
| `#ff6600` | **Ярко-оранжевый** | Предупреждения |
| `#ffffff` | Белый | Текст |
| `#b0b0b0` | Серый | Вторичный текст |

### Градиенты

```css
/* Зелёный (Adidas) */
--primary-gradient: linear-gradient(135deg, #39ff14 0%, #2db810 100%);

/* Оранжевый (Предупреждение) */
--warning-gradient: linear-gradient(135deg, #ff6600 0%, #cc5200 100%);

/* Красный (Опасность) */
--danger-gradient: linear-gradient(135deg, #ff3333 0%, #cc0000 100%);
```

## 🔧 Настройка

### Изменить зелёный цвет:
```css
:root {
    --neon-green: #YOUR_COLOR;
}
```

### Изменить оранжевый:
```css
:root {
    --neon-orange: #YOUR_ORANGE;
}
```

```css
.hero-section {
    box-shadow: 0 0 40px rgba(57, 255, 20, 0.6);
}
```

```css
/* Удалить класс .glitch:hover */
```
