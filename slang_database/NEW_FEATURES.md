## 5️⃣ Случайное слово

**Файлы:**
- `app.py` - маршрут `/random`
- `templates/index.html` - кнопка на главной

**Как работает:**
```python
@app.route('/random')
def random_word():
    word = Word.query.order_by(func.random()).first()
    return redirect(url_for('word_detail', word_id=word.id))
```

**Демонстрация:**
1. На главной странице нажмите **"Случайное слово"** (жёлтая кнопка)
2. Или перейдите на http://localhost:5000/random

## 7️⃣ История изменений

**Файлы:**
- `app.py` - логирование в `edit_word()`
- `templates/word_detail.html` - блок истории
- `models.py` - модель `WordHistory` (уже есть)

**Как работает:**
```python
# При редактировании сохраняем изменения
history = WordHistory(
    word_id=word.id,
    changed_by='Anonymous',
    changes='word: старый → новый, meaning: ...'
)
```

**Демонстрация:**
1. Откройте любое слово
2. Нажмите "Редактировать"
3. Измените значение
4. Сохраните
5. Внизу страницы появится **"История изменений"**

## 8️⃣ Поиск по синонимам

**Файлы:**
- `app.py` - обновлённый поиск в `/search`

**Как работает:**
```python
words_query = words_query.filter(
    db.or_(
        Word.word.ilike(f'%{query}%'),
        Word.meaning.ilike(f'%{query}%'),
        Word.synonyms.ilike(f'%{query}%')  # ← Поиск по синонимам!
    )
)
```

**Демонстрация:**
1. Откройте http://localhost:5000/search
2. Введите "друг" (синоним к "чувак")
3. Найдётся слово "чувак" через синонимы!
