## 1️⃣1️⃣ Таймлайн эволюции слова

**Файлы:**
- `app.py` - API `/api/word/<id>/evolution`
- `templates/word_detail.html` - визуализация таймлайна
- `static/js/main.js` - загрузка данных

**Как работает:**
```python
# Находит слова с похожим значением/синонимами
# в разных периодах и сортирует по времени
similar_words = Word.query.filter(
    db.or_(
        Word.word.ilike(f'%{word.word}%'),
        Word.synonyms.ilike(f'%{word.word}%'),
        Word.meaning.ilike(f'%{word.meaning[:30]}%')
    )
).order_by(Word.period_id).all()
```

**Визуализация:**
- Вертикальный таймлайн с точками
- Текущее слово выделено цветом
- Показывает десятилетие и значение

**Демонстрация:**
1. Откройте http://localhost:5000/word/1
2. Прокрутите до блока **"Эволюция слова"**
3. Увидите как слово менялось со временем
