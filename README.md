# База данных молодежного сленга в России

Курсовая работа по созданию веб-приложения для исследования эволюции русского молодежного сленга с 1920-х годов до наших дней.

## 📋 Описание проекта

Приложение представляет собой веб-интерфейс для управления базой данных сленговых выражений разных периодов времени. Позволяет добавлять, редактировать, искать и анализировать сленг различных десятилетий.

## ✨ Основные возможности

- **Управление словами**: добавление, редактирование, удаление
- **Поиск и фильтрация**: по слову, значению, синонимам, периодам, категориям, тегам
- **Структурирование**: периоды (десятилетия), категории, теги
- **Синонимы и антонимы**: расширенная лингвистическая информация
- **Голосование**: система рейтинга слов
- **Комментарии**: обсуждение значений
- **Статистика**: визуализация данных с помощью графиков
- **Экспорт**: выгрузка данных в CSV и JSON форматах
- **API**: RESTful API для интеграции

## 🛠️ Технологии

- **Backend**: Python 3.9+, Flask 3.0
- **Database**: SQLite / PostgreSQL (через SQLAlchemy)
- **Frontend**: HTML5, CSS3, Bootstrap 5, Chart.js
- **Аутентификация**: Flask-Login
- **Формы**: Flask-WTF

## 📦 Установка

### Требования

- Python 3.9 или выше
- pip

### Быстрый старт

```bash
# 1. Перейдите в директорию проекта
cd slang_database

# 2. Создайте виртуальное окружение
python -m venv venv

# Windows:
venv\Scripts\activate

# Linux/Mac:
source venv/bin/activate

# 3. Установите зависимости
pip install -r requirements.txt

# 4. Настройте переменные окружения
# Скопируйте .env.example в .env и измените при необходимости
copy .env.example .env

# 5. Инициализируйте базу данных
python init_db.py

# 6. Запустите приложение
python app.py
```

Приложение будет доступно по адресу: http://localhost:5000

## 🗄️ Модели данных

### Основные сущности

- **User** — пользователи системы (администраторы, модераторы, пользователи)
- **Word** — слова сленга (основная сущность)
- **Period** — временные периоды (десятилетия)
- **Category** — категории слов
- **Tag** — теги для гибкой классификации
- **Source** — источники информации
- **Comment** — комментарии к словам
- **Vote** — голосование за слова
- **WordHistory** — история изменений слов

### Связи

- Word ↔ Period: Many-to-One
- Word ↔ Category: Many-to-One
- Word ↔ Tag: Many-to-Many
- Word ↔ User: One-to-Many (автор)
- Word ↔ Source: One-to-Many
- Word ↔ Comment: One-to-Many
- Word ↔ Vote: One-to-Many

## 🔍 Поиск и фильтрация

Поиск поддерживает:
- Полнотекстовый поиск по слову и значению
- Поиск по синонимам
- Фильтрация по периоду
- Фильтрация по категории
- Фильтрация по тегам
- Сортировка по алфавиту, популярности, дате

Примеры запросов:
- `/search?q=кринж` — найти слово "кринж"
- `/search?period=3&category=2` — слова категории 2 периода 3
- `/search?q=молодежный&tag=интернет` — поиск с тегом

## 📊 API endpoints

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/search` | Поиск слов (JSON) |
| GET | `/api/timeline` | Временная шкала слов |
| GET | `/api/popular-tags` | Популярные теги |
| GET | `/api/detailed-stats` | Подробная статистика |
| GET | `/api/statistics` | Базовая статистика |
| POST | `/api/vote` | Голосование за слово |
| POST | `/api/favorite/<id>` | Добавить в избранное |
| GET | `/export/csv` | Экспорт в CSV |
| GET | `/export/json` | Экспорт в JSON |

## 🎨 Визуализация

Статистика отображается с помощью Chart.js:
- Круговая диаграмма — распределение слов по периодам
- Столбчатая диаграмма — слова по категориям
- Линейный график — эволюция сленга по десятилетиям

## 🔐 Безопасность

- CSRF-защита форм (Flask-WTF)
